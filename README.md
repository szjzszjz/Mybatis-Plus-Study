# springboot-Mybatis-Plus-Study
### 通过视频学习MP的过程中写的demo  
#### 总结：基础的CRUD以及扩展的CRUD MP已经帮我们实现 一般不要做任何的配置即可满足业务的需求。 个人感觉和spring data jpa 同样方便好用，甚至有过之而无不及。  
#### 注：3.0.7以后MP 支持lambda表达式，并运用建造者模式使用链式调用使查询更新更加方便简洁。所有如果使用lambda，则要求jdk 8
